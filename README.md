\# CareerLens AI 🚀



CareerLens AI is a machine learning powered career intelligence platform that analyzes job listings and provides insights about:



\- Salary prediction

\- Career path recommendations

\- Skill demand forecasting

\- Job role matching

\- Job market analytics



\## Features



\- 📊 Interactive dashboard

\- 🤖 Machine learning models

\- 💰 Salary prediction

\- 🧠 Skill demand forecasting

\- 📍 Location-based job insights

\- 📈 Analytics hub with visualizations



\## Tech Stack



\- Python

\- Streamlit

\- Scikit-learn

\- XGBoost

\- Prophet

\- Plotly

\- Pandas

\- NumPy



\## Dataset



54K+ job listings dataset enriched with:



\- Skills

\- Salary

\- Company

\- Location

\- Experience



\## Run Locally



Install dependencies:

pip install -r requirements.txt





Run the Streamlit app:





streamlit run app.py
